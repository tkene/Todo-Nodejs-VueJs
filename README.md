# Todo Tags Project (minimal)

Contains a minimal backend (Node + Express) and frontend (Vue 3 + Vite).

Quick start:

1. Backend:
   - cd backend
   - npm install
   - npm start
2. Frontend:
   - cd frontend
   - npm install
   - npm run dev
3. Open http://localhost:5173 and check notes in frontend/README.md

This archive is designed for quick live-coding: add a feature such as tags, edit, or comments.
